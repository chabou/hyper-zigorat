![screenshot](https://cdn.rawgit.com/dotamir/hyper-zigorat/master/Screenshot.png)

## Install
Add `hyper-zigorat` to your plugins in `~/.hyper.js`, like so:

`~/.hyper.js`:
```javascript
plugins: [
  'hyper-zigorat'
]
```
